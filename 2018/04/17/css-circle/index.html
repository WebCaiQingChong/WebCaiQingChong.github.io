扇形，实现了一个扇形，那其他的几个进行旋转。

我们肯定知道利用css的border来实现三角形，那么如何实现一个等边三角形呢？

结合三角函数，以及border生成三角形，我们把等边三角形从顶角等分，分成两个30°顶角的直角三角形，三角形短边就是我们所设置的

就是r*cos(30°),这样我们的等边三角形就出来了(因为cos(30°)所得到的只能是一个近似值，所以这样做出来的肯定是有误差的)。

里面有几个需要注意的点：

  1. css transform-origin 定义旋转的中心点

  2. clip rect(top, right, bottom, left)，其中需要注意的是right和bottom，这里面的值都是基于左上角来算的。

  4. css transform: rotate() 基于某轴的旋转

总结
